# openwrt firmware buider for zbt-z8106ax-m2-t (adapted from w1700k builder)

This project automates the process of building OpenWrt firmware images for the zbtlink zbt-z8106ax-m2-t.

- forked from https://github.com/openwrt-fanboy/w1700k-ubi-build
- which was forked from https://github.com/glassd00r/Openwrt_Builder
- which was forked from https://github.com/JuliusBairaktaris/Qualcommax_NSS_Builder

fastbuild adapted from https://github.com/tete1030/openwrt-fastbuild-actions
